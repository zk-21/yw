四年级核心阶段

从“读懂内容”提升到“概括清楚、依据充分、表达有层次”

四年级语文的分水岭不在背了多少知识点，而在孩子能不能把答案说完整：先概括，再找依据，最后说明作用。阅读和作文要同步训练，才能真正提优。

四年级必须形成的 5 个能力

  • 概括能力：能用“人物 + 事件 + 结果”说清一篇文章主要内容。
  • 依据意识：阅读题不凭感觉，能回到原文找关键词和关键句。
  • 赏析能力：能说清修辞、描写、关键词好在哪里。
  • 表达能力：作文能把经过写具体，加入动作、语言、神态、心理。
  • 复盘能力：知道自己错在审题、概括、依据、表达还是书写。

提优总口诀

概括：谁做了什么，结果怎样。

依据：从“原文句子”可以看出。

赏析：方法 + 特点 + 效果。

作文：动作 + 语言 + 神态 + 心理。

复盘：错因 + 正确方法 + 同类再练。

字词基础

从“会认会写”到“会在语境中运用”

多音字

似：似乎 si / 似的 shi

系：关系 xi / 系鞋带 ji

降：下降 jiang / 投降 xiang

曲：弯曲 qu / 歌曲 qu

模：模糊 mo / 模样 mu

提优练法：每个读音必须放进句子里判断意思。

形近字

辩 / 辨 / 辫 / 瓣：辩论、辨别、辫子、花瓣。

厉 / 历：严厉、历史。

暮 / 幕 / 慕 / 墓：暮色、幕布、羡慕、坟墓。

载 / 栽 / 裁：载重、栽树、裁剪。

提优练法：先看偏旁，再看词义，最后放回句子检查。

词语辨析与搭配

类型例子训练重点
近义词逐渐/渐渐，居然/竟然，观赏/欣赏说清细微差别，不能随便替换
反义词宽阔/狭窄，热闹/冷清，清晰/模糊结合语境判断意思相反
形容词搭配灿烂的阳光，宽阔的马路，清新的空气避免“提高错误、养成态度”这类搭配不当
动词搭配改正错误，提高成绩，保护环境，珍惜时间积累作文中常用的准确表达

句子专项

把句子改准确、写具体、表达清楚

扩句

方法：时间 + 地点 + 样子 + 动作 + 心情。

例：小鸟唱歌。变成：清晨，几只可爱的小鸟在枝头欢快地唱歌。

缩句

方法：去掉修饰语，保留“谁做什么”或“什么怎么样”。

例：漂亮的小鸟在树枝上唱歌。缩成：小鸟唱歌。

句式转换

重点：把字句、被字句、陈述句、反问句意思不能变。

例：我们应该珍惜时间。变成：难道我们不应该珍惜时间吗？

修改病句常考类型

病因病句修改
成分残缺通过这件事，使我明白了道理。这件事使我明白了道理。
搭配不当我们要养成认真学习的态度。我们要养成认真学习的习惯。
重复啰嗦我估计他大概不会来了。我估计他不会来了。
前后矛盾全班同学都来了，只有小明没来。除了小明，全班同学都来了。
用词不当他穿着一顶帽子。他戴着一顶帽子。

阅读理解提优

四年级真正拉开差距的核心模块

阅读题型、答法与提优要求

题型答题模板常见失分提优要求
概括主要内容人物 + 事件 + 结果复述太长，只写开头语言简洁，保留结果
概括段意这一段主要写了……抄原文，不会提炼抓中心句和关键词
理解词语这个词在文中指……只写字典意思联系上下文解释文中意思
找依据从“……”可以看出……没有引用原文答案必须落回文本
人物形象从“……”看出他是一个……的人只写品质，不写事例品质 + 具体事例
句子赏析方法 + 特点 + 效果只写“很生动”说清哪里生动、为什么好
句子作用位置 + 内容作用 + 情感作用套模板不结合文章结合上下文说明作用

阅读短文一：蚂蚁搬家

夏天的午后，天空突然乌云密布，眼看就要下雨了。我坐在院子里乘凉，忽然发现地上有一群蚂蚁在忙碌地搬家。它们排着长长的队伍，有的背着食物，有的拖着幼虫，一个个匆匆忙忙地往高处爬。我想起妈妈说过，蚂蚁搬家常常预示着要下雨。果然，没过多久，大雨就倾盆而下。小小的蚂蚁真聪明，它们竟然能预知天气变化。

基础题：这篇短文主要写了什么？

提优题：从哪里可以看出蚂蚁搬家很有秩序？

拔高题：结尾“小小的蚂蚁真聪明”有什么作用？

高分答法：结尾总结全文，点明中心，表达了作者对蚂蚁能预知天气变化的惊奇和赞叹。

阅读短文二：我的同桌

我的同桌小雨有一双亮晶晶的眼睛，笑起来有两个甜甜的酒窝。她特别乐于助人。有一次，我忘记带铅笔盒，小雨毫不犹豫地把备用铅笔借给我。还有一次，我数学题不会做，她耐心地给我讲解，直到我听懂为止。小雨不仅学习好，还很关心同学，大家都很喜欢她。

基础题：这一段主要写了小雨的什么特点？

提优题：请结合文中事例分析小雨的人物形象。

拔高题：“毫不犹豫”体现了小雨什么品质？

高分答法：小雨是一个乐于助人的人。从她借铅笔、耐心讲题可以看出她关心同学、愿意帮助别人。

修辞赏析

从“判断修辞”升级到“分析表达效果”

比喻

例句：雨点儿落在窗户上，像一串串小鼓点。

低分答案：用了比喻，很生动。

高分答案：这句话用了比喻，把雨点落在窗户上的声音比作小鼓点，写出了雨声清脆、有节奏的特点，使句子更加生动。

拟人

例句：小草从土里探出头来。

低分答案：用了拟人，很好。

高分答案：这句话用了拟人，把小草当作人来写，写出了小草生长时可爱、有生命力的样子，让画面更鲜活。

修辞答题模板

修辞判断方法作用表达
比喻有“像、好像、仿佛”，把 A 比作 B写出了事物的形状、颜色、声音或特点，使表达生动形象
拟人把事物写成人，有人的动作或情感写出了事物的生命力，让画面更有趣
排比三个或三个以上结构相似的句子增强语气，使表达更有气势
夸张故意放大或缩小特点突出事物特点，增强表达效果
反问用问句表达肯定或否定加强语气，引起读者思考

人物描写

用细节让人物活起来

动作描写

抓住具体动作：攥、扶、推、捧、擦、盯、跑、跳。

例：他紧紧攥着试卷，手心里全是汗。

语言描写

人物说的话要符合身份和当时情境。

例：“别怕，我陪你一起去。”小明轻声说。

神态描写

写脸色、眼神、表情，表现人物心情。

例：她皱着眉，眼睛一动不动地盯着题目。

心理描写

写心里的想法，不只写“我很紧张”。

例：我的心像揣着一只小鼓，咚咚直响。

作文提优

四年级作文要写清楚一件事，并写出具体细节

写事作文结构

开头：交代时间、地点、人物、事情。

中间：写清经过，重点部分详写。

结尾：写感受、收获或点题。

提优重点：起因和结果略写，经过详写。

片段升级示例

普通写法：我参加比赛，很紧张，后来成功了。

提优写法：轮到我上场时，我紧紧攥着演讲稿，掌心全是汗。台下黑压压的一片，我的心怦怦直跳。想到老师鼓励我的话，我深吸一口气，终于开口读了起来。

作文类型与训练重点

类型写作重点提优训练
写人用一件事表现人物特点外貌抓特点，事例写具体，结尾写感受
写事六要素完整，经过详写加入动作、语言、神态、心理
写景按顺序写景物特点写颜色、声音、变化和自己的感受
观察作文写清观察对象和变化看、听、闻、想、感受都要有

古诗文

不只会背，还要会解释、会体会情感

学习四步法

第一步：读准字音和停顿。

第二步：理解重点词语。

第三步：说出诗句大意。

第四步：体会诗人情感。

常见情感

思念家乡、惜别朋友、赞美自然、表达志向、热爱生活、赞美人物品质。

答题模板：这句诗描写了……的画面，表达了诗人……的感情。

低分到高分

四年级提优最有效的训练方式

题目低分答案高分答案方法点
这句话好在哪里？很生动。用了比喻，把落叶比作蝴蝶，写出了落叶飘落时轻盈、优美的样子。方法 + 特点 + 效果
小明是怎样的人？小明很好。小明是一个坚持不懈的人，从他每天早起练跑步可以看出。品质 + 依据
文章主要写了什么？写小明跑步。文章写了小明坚持练习跑步，最后在运动会中取得好成绩的事。人物 + 事件 + 结果
作文写紧张我很紧张。我紧紧攥着衣角，眼睛盯着试卷，手心里全是汗。用动作和神态表现心情

每周训练

按能力安排，而不是随便刷题

时间训练内容目标
周一多音字、形近字、词语搭配减少基础失分
周二扩句、缩句、句式转换、病句句子表达准确
周三一篇阅读：概括、词语、依据题答案回到原文
周四修辞赏析和仿写说清表达效果
周五作文片段：动作、语言、神态、心理把片段写具体
周六综合阅读 + 作文提纲整合能力
周日错题复盘 + 同类再练形成方法，不重复错

家长陪学

四年级不要只问“答案是什么”

阅读追问

你从文章哪里看出来？能不能把那句话圈出来？

作文追问

这件事最重要的地方在哪里？能不能把动作和心情写出来？

错题追问

你错在没审清题，还是没找到依据，还是答案没写完整？
