练习计划

本年级最容易掉分的 3 件事

  • 说明文：知道说明方法，却答不出“说明了对象的什么特点”。
  • 读后感：复述故事太多，自己的感受、联系和启发太少。
  • 中心详略：作文材料不少，但没有围绕中心安排详写和略写。

补救入口

  • 说明文：按“方法 + 特点 + 作用”写答案。
  • 读后感：用“少复述、多感受、联系生活”检查。
  • 作文：先写中心句，再决定哪一段详写。

本页使用顺序

  • 先看目标：确认主题、人物、结构、说明文和作文中心是否稳。
  • 再看题型方法：阅读答案要从一句话升级成分层表达。
  • 接着看五升六衔接：提前练综合阅读、非连续文本和限时表达。
  • 最后做复盘：用练习计划改写低分答案，留下错因和下次提醒。

家长检查清单

  • 阅读：答案是否有观点、依据和分析，不只写结论。
  • 说明文：能不能说清说明方法和语言准确性。
  • 作文：选材是否围绕中心，重点段是否有细节。
  • 立意：结尾是否写出变化、发现或思考。

五年级低分到拔尖样例

任务普通答案拔尖答案得分点
说明文语言“大约”不能删，删了不准确。“大约”表示估计，不是完全确定。删去后就变成非常肯定的数量，与实际情况不一定相符，体现了说明文语言的准确性。词义、删后变化、准确性三层完整。

五年级必须形成的 5 个能力

  • 主题分析：能概括文章中心思想，而不是只说“很感人”。训练时先概括事件，再追问“作者想表达什么”。
  • 人物分析：能结合语言、动作、心理、事例分析人物形象。答题固定写“特点 + 原文依据 + 具体分析”。
  • 结构意识：能判断开头、过渡、照应、结尾的作用。先看位置，再分别想内容作用和结构作用。
  • 说明文能力：能判断说明方法，分析语言准确性。每次都说清“说明对象的什么特点”。
  • 写作能力：能围绕中心选材，安排详略，结尾点题。写前先写一句中心，再圈出最该详写的一幕。

提优总口诀

中心：主要内容 + 作者情感或道理。

人物：特点 + 原文依据 + 具体分析。

作用：位置 + 内容 + 情感或结构作用。

说明：方法 + 说明对象特点 + 表达效果。

作文：中心明确，详略得当，首尾照应。

阅读理解

五年级记叙文模板怎么用

题型满分得分点满分表达常见失分
中心思想内容 + 情感 + 道理文章通过写……，表达了……，告诉我们……。只写内容，不写情感或道理。
人物形象特点 + 事例 + 分析从“……”可以看出他是一个……的人，因为……表现了他……。只写品质，没有依据。
环境描写环境特点 + 人物心情 + 情节/中心这里写出了……的环境，烘托了……，为下文……作铺垫/突出……。只写“写天气”。
句段作用内容作用 + 结构作用 + 情感作用内容上写了……；结构上……；情感上表现了……。生套模板。
标题作用内容 + 线索/悬念 + 中心题目“……”概括了……，也是……的线索，点明/暗示了……。答案空泛。
词句含义表层意思 + 深层含义 + 中心表面上指……，实际上写出了……，表现了……。只解释字面意思。

记叙文训练：父亲的背影

那年冬天，父亲送我去车站。雪下得很大，地上积了厚厚一层。父亲提着我的行李箱走在前面，我跟在他身后。他的背影在雪地里显得有些单薄，但步伐却很坚定。到了车站，父亲帮我安顿好座位，又给我买了热腾腾的包子。车要开了，父亲站在站台上向我挥手，雪花落在他的头发上，他却浑然不觉。看着父亲渐渐远去的背影，我的眼眶湿润了。

基础题：文章写了一件什么事？

提优题：文中的父亲是一个怎样的人？请结合原文分析。

拔高题：文章多次写“雪”有什么作用？

满分答法：写雪交代了寒冷的环境，烘托出父亲送“我”时的辛苦，也更能突出父爱的深沉。

人物形象答题示范

低分答案：父亲很好，很爱我。

满分答案：父亲是一个关爱孩子、默默付出的人。从他冒雪送“我”去车站、帮“我”安顿座位、买热包子可以看出，他把孩子放在心上，自己辛苦却不说出来。

方法点：人物特点不能只贴标签，必须用原文事例支撑。

说明文阅读

说明文重在准确、清楚、有条理

五年级说明文考点怎么用

考点判断方法作用答法
举例子有“比如、例如、如”等词通过具体例子说明事物特点，使说明更具体
列数字出现具体数字用准确数字说明特点，使说明更准确
作比较把两个事物进行对比突出说明对象的特点
打比方用比喻来说明使抽象内容更形象易懂
分类别按类型分开说明使说明更有条理
语言准确性“大约、通常、可能、左右”等词体现说明文语言准确、严谨

说明文训练：松鼠

松鼠是一种漂亮的小动物，乖巧、驯良，很讨人喜欢。它们面容清秀，眼睛闪闪发光，身体矫健，四肢轻快。松鼠喜欢在树上活动，窝通常搭在树枝分叉的地方，又干净又暖和。松鼠主要吃松果、榛子等坚果，也会把食物藏在树洞里，留着冬天吃。松鼠一胎通常生三四个幼崽。

基础题：这段文字主要介绍了松鼠哪些方面？

提优题：“通常”能不能删去？为什么？

拔高题：这段说明语言有什么特点？

满分答法：“通常”不能删，表示一般情况，不排除特殊情况，体现了说明文语言的准确性。

说明方法答题示范

题目：“松鼠一胎通常生三四个幼崽”用了什么说明方法？有什么作用？

满分答案：用了列数字的说明方法，用“三四个”准确说明松鼠一胎幼崽的数量，使说明更具体、更准确。

古诗词赏析

五年级古诗要能分析意象、情感和表达手法

五年级古诗词情感怎么用

情感类型常见意象分析方向答题表达
思乡怀人明月、秋风、故园、家书借景抒情借……表达诗人对家乡或亲人的思念。
送别惜别柳、酒、孤帆、长亭、远影景中含情通过送别场景表达依依不舍和祝福。
热爱自然春风、花鸟、山水、田园动静结合描写自然景色，表达喜爱和赞美。
托物言志梅、竹、松、石灰、荷花借物写人借……表现诗人高洁、坚强或不屈的品质。
忧国忧民战争、百姓、边塞、寒夜联系背景表达诗人对国家命运或百姓疾苦的关心。
人生哲理登高、流水、山路、明月由景悟理通过景物描写表达某种人生道理。

古诗赏析四步法

第一步：抓意象。看诗中写了哪些景物或事物。

第二步：想画面。把景物连成一幅画面。

第三步：辨情感。判断诗人是思念、赞美、惜别，还是表达志向。

第四步：说手法。常见有借景抒情、托物言志、动静结合、对比。

古诗满分判定标准

画面要具体：写出景物、状态和整体氛围。

手法要点明：借景抒情、托物言志、动静结合、对比等要判断准确。

情感要落准：结合意象和语境，不只写“喜欢/伤心”。

满分模板：诗人通过描写……的画面，运用……的手法，借……表达了……的感情。

炼字模板：“……”字写出了……的特点，使画面更加……，也表现了诗人……的心境。

低分答案到满分答案

题目低分答案满分答案
表达了什么情感？表达了思乡。诗人借明月这一意象，写自己望月时想到远方家乡，表达了浓浓的思乡之情。
“绿”字好在哪里？写出了绿色。“绿”字写出了春风吹过后江南大地草木变绿的景象，使画面鲜明、有生机。
托物言志是什么意思？写东西。诗人表面写某种事物，实际借它表现自己的志向、品格或精神追求。

诗人、词人表达情感的高级路径

表达路径核心理解常见内容满分答题表达
借景抒情通过景物写心情月、风、雨、雪、落日、江水诗人/词人借……的景象，含蓄地表达了……的情感。
情景交融景物和情感融合在一起寒雨、孤舟、残阳、秋风景中含情，……的景象烘托出诗人/词人……的心境。
托物言志借事物表现人格或志向梅、兰、竹、菊、松、石灰表面写……，实际表现诗人……的品格或追求。
用典抒情借历史人物或故事表达情感古人、典故、历史事件借……的典故，表达诗人/词人……的志向或感慨。
对比衬托用差异突出情感今昔对比、动静对比、乐景写哀通过……和……的对比，突出诗人/词人……的情感。
炼字传情一个关键字承载画面和情绪绿、破、惊、满、愁、孤“……”字写出了……的特点，也表现了诗人/词人……的心情。

诗与词的情感表达区别

诗：常借景、借物、叙事表达情感，语言凝练，画面感强。

词：更常写细腻心绪，如离愁、思念、感伤、豪情，节奏和语气更委婉。

答题提醒：遇到词，不只看写了什么景，还要看词人内心的变化。

完整答题模板

第一句：这首诗/词描写了……的画面。

第二句：诗人/词人运用了……的手法。

第三句：借……表达了……的情感。

示例：诗人借明月这一意象，描写月夜思乡的画面，表达了远离家乡时对亲人的思念。

特级教师拔尖提优

五年级从“会做题”提升到“会分析、会表达”

五年级拔尖要训练答案深度和作文立意，为六年级综合阅读、作文升格和小升初表达打基础。

1. 阅读要答出层次

中心思想、人物形象、环境描写作用不能只答一句，要形成“内容 + 手法 + 作用”的完整表达。

拔尖训练：每道分析题至少写两层意思，并补充文本依据。

2. 说明文要抓准确性

五年级常考说明方法和说明语言，尤其是“能否删去、为什么”。

拔尖训练：按“词义 + 删去后变化 + 体现准确性”组织答案。

3. 作文要有中心和详略

满分作文不是写得长，而是中心明确、材料集中、重点片段细致。

拔尖训练：同一题目列两个立意，选更具体、更有变化的一个写。

五年级拔尖训练路径

  • 基础达标：能概括文章内容，能判断说明方法，作文结构完整。
  • 能力提升：能分析人物形象、句段作用、环境描写作用。
  • 拔尖发展：能从主题、结构、语言三方面组织高质量答案。

五年级每日拔尖 15 分钟

4 分钟：读一段文章，写一句内容概括。

4 分钟：选一句关键句，分析它对人物或中心的作用。

4 分钟：改写一个低分答案，让它有依据、有层次。

3 分钟：积累一个作文素材，并写出可表达的中心。

年级衔接

五升六衔接：从“专项会做”到“综合运用”

五年级已经接触主题、结构、说明文和文言启蒙，升六后要把这些能力合在综合题里使用。衔接重点是限时、整合和复盘。

综合阅读

五年级分题型训练，升六要能一篇文章里同时处理概括、人物、句段作用和中心。

非连续文本

提前练图表和材料阅读：先找信息，再比较差异，最后写结论或建议。

作文冲刺

五年级练中心和详略，升六要增加审题速度、点题意识和结尾升格。

每周安排

每周 1 次综合阅读、1 次说明文或非连续文本、1 次作文片段升格。

环境描写作用环境特点 + 人物心情 + 情节/中心这里写出了……的环境，烘托了……，推动了……/突出……。结合人物心情、情节发展或中心答。
说明方法作用方法 + 对象特点 + 表达效果这里用了……的说明方法，具体/准确/形象地说明了……的特点。不同说明方法作用不同，不能一律写“生动”。

作文升格前后变化 + 重点细节 + 成长思考这件事让我从……变成……，我明白了……。结尾要有变化和思考，不只喊口号。

拔尖题型示范

五年级普通答案怎么升格

题目普通答案拔尖答案得分点
文章表达了怎样的中心思想？文章很感人。文章通过写父亲冒雪送“我”去车站的事，表现了父亲对子女深沉而含蓄的爱。内容 + 情感
环境描写有什么作用？写天气冷。这里写雪大、风冷，交代了恶劣环境，衬托出父亲送行的辛苦，也更突出父爱的深沉。环境 + 人物 + 中心
“大约”一词能删去吗？不能，删去不准确。不能。“大约”表示估计，不是准确数字，删去后变成肯定说法，与事实可能不符，体现说明文语言的准确性。词义 + 删后变化 + 准确性
作文怎样写出成长？写我学会了骑车。可以写“我”从害怕摔倒到敢于尝试，最后明白遇到困难不能轻易放弃，突出心理变化和成长收获。变化 + 过程 + 感悟
