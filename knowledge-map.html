模板速查

小学语文常见题型 · 答题思路与模板总表

孩子做题时先看“思路”，再套“模板”。模板不是死背，而是提醒答案要包含哪些得分点。

题型答题思路答题模板提分技巧
概括主要内容先找人物、事件、结果，高年级再看中心。文章写了……做……，结果……，表现了……。不要只写开头事件，要把结果和变化写进去。
理解词语含义先解释本义，再结合上下文说文中义。“……”本来指……，文中指……，写出了……。不能离开原文解释词语。
句子赏析找修辞、描写方法或关键词，再说效果和情感。这句话运用了……，写出了……，表达了……。必须点出“哪个词/哪种手法”好在哪里。
人物形象从语言、动作、神态、心理和事件找依据。他/她是一个……的人，从“……”可以看出。特点不能空写，必须带原文依据。
说明方法作用先判断方法，再说说明了对象的什么特点。这句话运用了……的说明方法，具体/准确/生动地说明了……。列数字答“准确具体”，打比方答“生动形象”。
说明文词语能否删解释词义，比较删前删后，回到准确性。不能删。“……”表示……，删去后变成……，体现说明文语言准确性。不要只写“不能删，不准确”，要说明为什么。
古诗画面题抓景物、颜色、动作和感受。我仿佛看到……，感受到……。不要只翻译诗句，要把画面说出来。
古诗情感题先看意象，再判断情感。诗人通过……，表达了……的感情。月亮常见思乡，杨柳常见送别，边关常见报国。
文言翻译逐词解释，再调成通顺现代汉语。先解释重点字：……；整句意思是……。注意古今异义，如“走”常是跑，“亡”可指丢失。
口语劝说先礼貌称呼，再说明理由，最后提出建议。……，我知道你……，但是……，我们可以……，好吗？语气不能硬，理由要具体。
通知写清时间、地点、人物、事情、要求。请……于……到……参加……，请带好……。缺一个要素就容易丢分。
作文审题判断题型，圈题眼，定中心，选材料。我要写……，中心是……，重点写……这一幕。写前先列三行提纲，避免跑题和流水账。
作文重点段把最能表现中心的一幕写慢。动作 + 语言 + 神态/心理 + 环境/结果。重点段至少占全文三分之一。
词语辨析比较语气、对象、范围和感情色彩。“……”适合……语境；“……”适合……语境。近义词不能只看意思，还要看搭配和情感。
标点修改先判断语气，再看提示语位置。谁说：“……” / “……”谁说。/ “……，”谁说，“……”。直接引用的句末标点通常放在引号内。

学习路径

按年级逐层掌握

一二年级：打地基

拼音、识字、写字、朗读、看图说话。重点是读准、写对、说完整。

三年级：会表达

多音字、形近字、扩句、自然段、中心句、观察日记、写完整一件事。

四年级：会概括

概括主要内容、联系上下文、修辞作用、人物描写、阅读找依据。

五六年级：会分析

中心思想、人物形象、说明文、文言文、作文详略、点题和结构。

特级教师拔尖辅导总纲

这张图谱不只用来背知识，更要用来诊断和训练

家长和孩子使用时，先看年级目标，再查薄弱题型，最后按训练闭环练。这样知识点才会真正变成分数。

先定位年级目标

一二年级先完整，三年级先成篇，四年级练具体，五六年级练中心和综合表达。

再判断题型

阅读先判断考概括、词句、人物、中心还是作用；作文先判断写人、写事、写景还是读后感。

最后形成闭环

每次训练都要有诊断、方法、例题、练习、自查、错题复盘，不能只刷题不复盘。

拔尖辅导检查

内容够不够，不看多少页，看有没有这 6 件事

拔尖辅导不是简单加难题，而是让孩子知道自己在哪一层、下一步怎么升格、练完怎么复盘。

检查项合格内容拔尖内容去哪里看
目标分层知道本年级必须掌握什么。能区分基础达标、能力提升、拔尖发展三层目标。各年级页“拔尖训练路径”
错因诊断知道题错了。能判断错在审题、知识点、依据、表达、格式还是复盘。易错诊断 / 诊断测评
方法示范给答案模板。给低分答案、问题分析、升格答案和满分标准。阅读模板 / 作文专题
迁移训练练同一道题。把同一种方法迁移到新文本、新题型、新作文题。练习计划
年级衔接只看当前知识点。知道从本年级到下一年级，能力要求怎样升级。各年级页“年级衔接”
复盘闭环订正答案。写清错因、正确方法、下次提醒，并做一次同类题。训练闭环

低年级拔尖

不提前刷难题，重点看拼音识字是否进入句子，表达是否完整、有顺序、有细节。

怎么带：每天 1 个字组词造句，1 幅图按“谁在哪里做什么”说完整。

中年级拔尖

重点看自然段、概括、依据和片段作文，能不能从“写出来”升级到“写具体”。

怎么带：每篇短文圈中心句，每个片段至少补动作、语言、心理中的两项。

高年级拔尖

重点看答案层次、中心立意、综合阅读和限时表达，能不能把方法迁移到新题。

怎么带：每道阅读题写“结论 + 依据 + 分析”，每篇作文先列中心和重点段。

模块一

分年级学习目标：从基础达标到拔尖提优

年级必须掌握提优目标怎么指导易错点每周训练
一年级拼音、识字、写完整句。看图能说清楚，写5-8句连贯话。拼音每天听、拼、读三遍；句子按“谁 + 做什么 + 在哪里”补完整。拼音混淆、句子不完整、标点漏写。拼读10分钟 + 看图说话1幅。
二年级词语积累、句子扩写、写一段话。多幅图能连成完整故事。先说图意顺序，再补时间、地点、动作、心情，最后检查标点。图意漏写、顺序乱、对话缺少提示语。看图写话2篇 + 词语造句10个。
三年级自然段、中心句、基础作文。能完整写一件事，有开头中间结尾。每段先找“这段主要写什么”，写作文先列起因、经过、结果。作文流水账，重点段太薄。句段训练3次 + 作文提纲2个。
四年级概括内容、修辞作用、人物描写。重点段能写具体，阅读答题有依据。概括用“谁做什么结果怎样”，赏析用“方法 + 特点 + 效果”。概括不准，阅读答案脱离原文。阅读题型4类 + 片段描写2段。
五年级中心思想、详略安排、说明文。答案有层次，作文有中心和详略。阅读答案先写结论再补依据；作文先定中心，再决定哪一段详写。作文立意浅，读后感复述太多。阅读模板训练 + 作文升格1篇。
六年级综合阅读、文言文、作文冲刺。能应对小升初综合题和限时作文。按分值分点作答；限时作文先用 5 分钟审题、列中心和重点段。答题不完整，作文点题不明显。真题模拟 + 错题复盘 + 满分答案改写。

模块二

阅读理解题型体系：按题型答，按分点写

阅读题型判断标志满分答题模板满分示例
概括主要内容题干出现“主要写了什么、概括内容”。谁 + 做了什么 + 结果怎样；高年级再加表达了什么。文章写了小男孩克服害怕参加演讲并获得掌声的事，表现了他勇敢尝试的变化。
理解词语含义题干问“这个词在文中指什么”。本义 + 文中义 + 表达效果。“暖流”本指温暖的水流，文中指老师的话让我感到安慰，写出了老师鼓励带来的力量。
句子赏析题干问“好在哪里、有什么表达效果”。手法/描写 + 写出了什么 + 表达了什么情感。这句话运用比喻，把落叶比作小船，写出落叶轻轻飘动的样子，表达了作者对秋天的喜爱。
人物形象题干问“他是一个怎样的人”。性格特点 + 原文依据 + 简要评价。他是一个细心的人。从他反复检查同学的作业本可以看出，他做事认真负责。
中心思想题干问“表达了什么、告诉我们什么”。文章写了……，表达了……，告诉我们……。文章写了作者种豆失败后继续观察的过程，表达了坚持探索的精神。
标题作用题干问“标题有什么妙处”。点明对象/线索 + 概括内容 + 暗示中心。“那盏灯”既点明写作对象，又贯穿全文，还暗示妈妈默默陪伴的爱。
环境描写作用题干问“这处景物描写有什么作用”。交代背景 + 烘托心情 + 推动情节/突出中心。雨越下越大，烘托了“我”焦急的心情，也为妈妈冒雨送伞作铺垫。
段落作用题干问“开头/结尾/中间段有什么作用”。开头：引出下文、设置悬念；中间：承上启下；结尾：总结全文、点明中心。结尾再次写掌声，照应开头，也点明“勇敢尝试会带来成长”的中心。
非连续性文本图表、通知、材料组合题。提取信息 + 比较数据 + 得出结论/提出建议。从表格看，阅读时间越长，阅读成绩越高。因此建议每天保持稳定阅读。

模块三

作文方法卡：拿到题目先知道从哪些方面写

作文类型写作抓手结构模板重点段失分提醒
写人作文写谁，什么特点，哪件事能证明。特点入题 → 典型事例 → 细节描写 → 品质影响。人物说话、动作、神态的具体场景。不要写成外貌和优点清单。
写事作文这件事让我发生什么变化。起因略写 → 经过详写 → 转折变化 → 结果感悟。最紧张、最后悔、最感动或最醒悟的一刻。不要从早写到晚，详略必须分明。
写景作文观察顺序和情感。总写印象 → 按顺序写景 → 特写镜头 → 抒发感情。颜色、声音、气味、动态和自己的感受。不要罗列景物名称。
状物作文物的特点以及它和我的关系。来历/总印象 → 外形特点 → 用途故事 → 情感意义。物品与“我”发生联系的一件小事。不要写成商品说明。
想象作文大胆想象但逻辑合理。设定前提 → 遇到问题 → 解决冲突 → 得到启示。问题怎么出现，主人公怎么解决。不要只写神奇，没有中心。
读后感少复述，多写感受和联系实际。简述内容 → 提出感受点 → 联系生活 → 写出改变。联系自己的真实经历。不要把读后感写成故事梗概。
半命题作文补题决定中心。补好题目 → 定中心 → 写关键事件 → 结尾扣题。补题关键词对应的场景。题目补得太大，文章就容易空。
普通写法问题分析升格写法教师点评
妈妈很爱我。太空，没有事件和画面。那天晚上我发烧，妈妈一会儿摸我的额头，一会儿看体温计，连拖鞋穿反了都没有发现。用动作细节表现爱，比直接说“爱我”更有说服力。
我很紧张。只是情绪标签。我的手心全是汗，心跳得像要从嗓子眼蹦出来，连稿纸上的字都开始晃动。心理描写可以通过身体反应写出来。
公园很美。没有具体景物。湖边的柳枝垂到水面上，风一吹，水面便皱起一圈圈细细的波纹。写景要把“美”变成可看见的画面。

模块四

古诗文提优：从会背到会赏析

模块补充内容答题模板/训练方法
古诗赏析景物、画面、情感、手法。这句诗描写了……的画面，运用……手法，表达了诗人……的情感。
常见情感思乡、送别、爱国、田园、童趣、惜时。先找景物和人物动作，再判断情感，不能只背诗意。
文言字词之、其、曰、乃、亡、善、走、顾、弗、矣。结合注释和上下文解释，注意古今异义。
文言断句先找人物和动作，再看语气词。谁 / 做什么 / 结果怎样。人物换了，常常要停顿。
翻译方法留、换、补、调。人名地名保留，古今不同的词换成现代说法，省略内容补出来，语序调顺。

模块五

基础知识易错诊断：知道为什么错，才知道怎么改

模块常见错法正确方法自查口诀
拼音n/l、平翘舌、前后鼻音混淆。放进词语和句子里读，不孤立背音节。先读准，再组词，最后造句。
字词形近字只看字形，不看偏旁意思。用偏旁判断意思：清和水有关，晴和太阳有关，睛和眼睛有关。看偏旁，想意思，放句子。
扩句缩句缩句把主干删掉，扩句堆词不通顺。缩句保留谁做什么；扩句补时间、地点、样子、动作。主干不能丢，修饰要合理。
修改病句读不出成分残缺、搭配不当、重复啰嗦。先找主干，再看搭配，最后删重复。读一遍，找主干，看搭配，删重复。
标点符号对话不会分段，引号冒号混用。人物说话要看提示语位置，句末标点放在引号内。谁说话，谁提示，标点跟着语气走。
病句类型判断方法示例修改提示
成分残缺看谁做什么是否完整。通过这件事，使我明白了道理。删“通过”或“使”，保留一个主语。
搭配不当看词语能不能搭配。我穿着漂亮的帽子。帽子要“戴”，衣服才“穿”。
重复啰嗦看意思是否重复。我忍不住情不自禁地笑了。“忍不住”和“情不自禁”保留一个。
前后矛盾看时间、数量、意思是否冲突。全班差不多都到齐了。“差不多”和“都”矛盾，删一个。

模块六

训练闭环：诊断、方法、例题、练习、自查、复盘

1. 诊断

先看孩子错在哪里：审题、知识点、表达、格式还是粗心。

2. 方法

只教一个方法，例如概括公式、重点段写法或病句四步法。

3. 例题

用一道典型题示范完整过程，让孩子知道为什么这样做。

4. 练习

同类题练3-5道，不追求数量，追求方法稳定迁移。

5. 自查

用清单检查：题型对不对、依据有没有、表达完整不完整。

6. 复盘

写清错因、正确方法、下次提醒，错题才真正变成能力。

问题表现说明孩子卡在哪训练方法
阅读题总答不到点。不会判断题型，答案缺依据。每天练3道题型判断，答案必须写原文依据。
作文像流水账。不会详略安排，不知道重点段写什么。每周练1个重点段，把关键一刻写到150字以上。
句子不通顺。语感和病句基础薄弱。每天朗读5分钟，修改3个病句。
古诗理解不会写。不会从画面过渡到情感。按“景物-画面-情感”口头说一遍，再写答案。

补充一

口语交际与综合性学习：说得清楚，也要说得得体

题型常考要求答题/表达模板易错提醒
转述把别人的话准确告诉第三个人。谁让谁在什么时候、什么地点做什么。注意人称变化，“我、你、他”要换准确。
劝说态度礼貌，理由充分，让对方愿意接受。称呼 + 表明理解 + 说明理由 + 提出建议。不能命令别人，也不能只说“这样不好”。
推荐推荐一本书、一个地方、一项活动或一位同学。我推荐……，理由有三点：……推荐理由要具体，不能只说“很好、很有趣”。
采访围绕主题设计问题，问题有顺序。您好，我想了解……请问……？谢谢您的回答。问题不要太空，要能问出具体信息。
通知写清时间、地点、人物、事情、要求。请……于……到……参加……，请带好……。时间地点不能漏，语气要简明正式。
讨论发言表达观点，有理由，能回应别人。我同意/不同意……，因为……，所以我建议……。不能只有观点没有理由，也不要偏离话题。
活动方案围绕主题安排步骤、分工和成果。主题：……；步骤：……；分工：……；展示方式：……。方案要能执行，不能只写口号。
材料整合从几则材料中提取共同信息。材料一说明……，材料二补充……，我得出的结论是……。不要照抄材料，要概括共同点。

补充二

书写规范与查字典：低年级打地基，高年级防丢分

模块必须掌握操作方法检查标准
笔画笔顺横、竖、撇、捺、点、提、折、钩等基本笔画。先横后竖，先撇后捺，从上到下，从左到右，先外后内再封口。能按正确顺序书空，写字不倒插笔。
偏旁部首常见部首的名称和意义。用偏旁猜字义：三点水多和水有关，提手旁多和动作有关。看到形近字能说出偏旁差别和意思差别。
间架结构左右、上下、包围、半包围、独体字结构。左右结构看宽窄，上下结构看高低，包围结构先外后内。字写在格子中间，大小匀称，不偏不挤。
音序查字法知道音节，先查大写字母，再找音节。先确定读音，再找音序，再查音节和页码。适合会读不会写、想查字形的情况。
部首查字法不知道读音时，先找部首。确定部首，数部首笔画，再数剩余笔画。适合不会读的生字，能查到读音和意思。
易错字纠错同音字、形近字、笔画易错字。错字本按“错字-正确字-组词-造句”记录。同一个错字连续三次写对，才算真正掌握。

补充三

标点符号专项：标点不是装饰，是语气和结构

标点什么时候用例句易错点
句号。一句话说完整，语气平稳。今天我去图书馆看书。不要一句到底不加句号。
逗号，一句话中间需要停顿。早上，我背着书包去上学。不能在主语和谓语之间乱停顿。
问号？表示疑问、反问。你今天为什么迟到？反问句虽然答案明确，句末也用问号。
感叹号！表示强烈语气、惊喜、赞叹、命令。这里的景色真美啊！不要每句话都用感叹号，语气会显得夸张。
冒号：提示下文、引出说话内容。老师说：“请大家安静。”提示语在后面时，一般不用冒号。
引号“ ”直接引用、特殊含义、特定称谓。妈妈说：“路上小心。”句末标点通常放在后引号里面。
省略号……内容省略、说话断断续续、语意未尽。我想说，可是……省略号是六个点，不要写成三个点。
破折号——解释说明、话题转换、声音延长。我最喜欢的地方——家乡的小河。不要和省略号混用。
分号；并列分句之间的较大停顿。春天花开了；夏天树绿了；秋天果熟了。低年级少用，高年级用于较长并列句。
书名号《》书名、文章名、作品名。我读了《西游记》。普通活动名称不用书名号。

补充四

阅读文体分类：先判断文体，再选择答法

文体阅读重点常考题型满分抓手
记叙文人物、事件、情节、中心。概括内容、人物形象、句段作用、标题作用。从原文找依据，围绕人物变化和中心作答。
写景散文景物顺序、画面、情感。景物特点、句子赏析、情感理解。抓景物词、修辞词、情感词。
说明文说明对象、特点、顺序、方法、语言准确性。说明方法作用、词语能否删、段落顺序。对象特点 + 方法作用 + 准确严谨。
童话人物拟人化、想象、道理。故事内容、人物特点、寓意启示。不能只复述故事，要写出明白的道理。
寓言短小故事背后的道理。寓意、人物行为评价、联系生活。故事内容 + 道理 + 现实生活例子。
科普文科学知识、说明顺序、术语准确。信息提取、说明方法、语言准确性。答案要忠于材料，不能凭常识乱答。
非连续性文本图表、通知、说明、材料组合。提取信息、比较数据、判断正误、提出建议。看标题、看数据、看单位、看条件。
古诗文阅读字词、画面、情感、道理。解释词语、翻译句子、赏析画面、概括启示。先读懂字词，再说画面和情感。

补充五

现代诗与儿童诗：读出画面、节奏和情感

常考题型答题思路答题模板提分提醒
描绘诗句画面找景物、颜色、动作和声音，把抽象诗句变成能看见的画面。我仿佛看到……，听到……，感受到……。不要只翻译字面意思，要写出“画面感”。
理解关键词先说词语本义，再结合诗中意象说它带来的感受。“……”本来指……，诗中写出了……，表现了……。儿童诗常用拟人和想象，不能只按普通词义解释。
体会诗人情感看反复出现的词、景物变化和语气变化。诗人借……表达了……的感情。情感要具体，如喜爱、思念、赞美、向往，不要只写“很开心”。
赏析修辞手法判断比喻、拟人、排比或反复，再说表达效果。这句运用了……，把……写得……，表达了……。反复不是重复啰嗦，常用来加强节奏和情感。
仿写小诗保留句式，换同类事物，情感要一致。原句写……，我可以换成……，仿写为……。仿写不是改几个字，要保持句式、意境和语气。
低分答案问题分析升格答案训练任务
这首诗写春天很美。只写结论，没有画面和情感依据。这首诗通过小草、花朵和春风等景物，写出了春天充满生机的画面，表达了作者对春天的喜爱。每读一首现代诗，圈出3个意象，再写一句“这些意象让我感到……”。
用了拟人，很生动。没有说清拟人把什么写活了。诗句把小溪当成人来写，说它一路唱着歌，写出了溪水流动时轻快活泼的样子，也表现了作者愉快的心情。找3句拟人句，按“把谁当成人 + 写出什么特点 + 表达什么情感”回答。

补充六

应用文与综合性学习：格式不丢分，表达能落地

类型必须写清格式/模板易错提醒
留言条给谁、什么事、谁留言、日期。称呼顶格写；正文写清事情；右下角写留言人和日期。不要漏写留言人，否则不知道是谁说的。
请假条向谁请假、原因、时间、请求批准、署名日期。我因……，需要请假……，请批准。原因要简明真实，时间范围要具体。
通知时间、地点、人物、事情、要求。请……于……到……参加……，请带好……。通知是给大家看的，语言要简洁正式。
倡议书倡议对象、原因、具体做法、号召。为了……，我倡议大家：一……；二……；三……。倡议不能只喊口号，要有能执行的做法。
建议书问题、原因、建议、期待。我发现……，建议……，这样可以……。语气要诚恳，不能像批评命令。
材料整合每则材料的信息、共同点、结论。材料一说明……，材料二补充……，因此我认为……。不要整段抄材料，要提炼共同信息。
图表题标题、单位、最大最小值、变化趋势。从图表可以看出……，因此建议……。答案要带数据依据，不能凭感觉下结论。

补充七

传统文化与文学常识：不死背，要知道怎么用

板块必须掌握常考方式训练方法
传统节日春节、元宵、清明、端午、中秋、重阳的时间和习俗。节日排序、习俗匹配、写一段节日介绍。按“时间 + 习俗 + 意义 + 我的经历”整理。
成语故事故事内容、人物做法、道理启示。解释成语、判断运用、联系生活谈启示。按“谁做了什么 + 结果怎样 + 告诉我们什么”复述。
寓言神话想象特点、人物愿望、故事道理。概括情节、评价人物、续写改写。找“愿望、困难、办法、结果、道理”五项。
名言警句学习、惜时、诚信、爱国、勤奋等主题。按语境填句、写作文引用。每句配一个适用场景，不只背原句。
作家作品课文作者、代表作品、人物形象或写作特点。文学常识选择、阅读背景理解。用“作者 + 作品 + 一个关键词”记忆。

补充八

古诗文体系：会背只是第一步，会赏析才是提优

类别常见内容判断方法答题表达
写景诗山水、春夏秋冬、风雨雪月。抓景物和颜色、动静变化。描绘了……的景色，表达了诗人对自然的喜爱。
送别诗朋友离别、渡口、长亭、杨柳。看“送、别、故人、归”等词。表达了诗人依依不舍和对友人的祝福。
思乡诗月亮、故乡、家书、游子。看“乡、家、月、归”等意象。表达了诗人思念家乡和亲人的感情。
爱国诗国家、边关、战争、报国。看国家命运、将士、壮志。表达了诗人热爱祖国、忧国忧民或报国之志。
田园诗村庄、农事、儿童、鸡犬、桑麻。看乡村生活和自然闲适。表现田园生活的宁静、自然和乐趣。
哲理诗登高、观物、变化、道理。从景物中悟出人生道理。告诉我们……的道理，启发我们……。
童趣诗儿童活动、游戏、观察。看孩子动作和天真想象。写出了儿童的活泼可爱和生活情趣。
文言文寓言故事、人物言行、道理启示。先找人物，再找动作和结果。这个故事告诉我们……，生活中我们也要……。
文言字词常见意思例句提醒学习方法
之他、它、的、去。结合句子判断，不要固定一个意思。放回原句翻译，看哪种意思通顺。
其他的、它的、其中。常指前文出现的人或物。先找“其”指代谁。
曰说。后面常接人物说的话。看到“曰”先找说话人。
乃于是、就、才。常表示事情接着发生。联系前后句理解。
亡丢失、逃跑、没有、死亡。不同课文意思不同。不能只记“死亡”。
善好、擅长。“善射”是擅长射箭。看后面接的是品质还是本领。
走跑。古文里常不是“走路”。注意古今异义。
顾看、回头看、拜访。要结合动作对象判断。把几个意思代入句子试一试。

补充九

词语专项：词语不是会背就行，关键是会用

专项学习重点训练方法易错提醒
近义词意思相近但语气、范围、对象可能不同。放进同一句话里比较哪个更准确。不能只看意思像不像，还要看搭配。
反义词意义相反或相对。按大小、多少、快慢、冷热、情感分类记。有些词没有绝对反义词，要结合语境。
词语搭配动词配名词，形容词配对象。积累“提高水平、改进方法、保护环境”等固定搭配。不能说“穿帽子、戴衣服”。
感情色彩褒义、贬义、中性。看这个词是夸人、批评人，还是客观描述。“狡猾”和“聪明”不能乱换。
多义词一个词在不同句子中意思不同。先解释本义，再说句中义。不能离开句子解释词语。
成语意思、出处、使用对象。每个成语至少造一个生活化句子。不要望文生义，例如“万人空巷”不是街上没人。
关联词并列、递进、转折、因果、假设、条件。先判断分句关系，再选关联词。“虽然……但是……”和“因为……所以……”不能混用。
俗语谚语歇后语生活经验和语言趣味。先理解意思，再联系生活例子。不要只背前半句，要知道适用场景。

专项提分实战库

从“知道考什么”升级到“知道怎么答、怎么练、怎么改”

下面每个模块都给出真实题型、低分问题、满分示范和训练任务。孩子可以照着练，家长也能直接判断哪里丢分。

练一题

先看题型，圈关键词，自己写一遍答案。

比答案

对照满分示范，看自己少了称呼、理由、依据、情感还是方法。

再迁移

同类题换一个材料再练一次，能迁移才是真掌握。

实战一

口语交际提分实战：语气得体，理由具体

题型真实题目低分答案满分示范训练任务
劝说同学在图书馆大声说话，你会怎样劝他？你别吵了！小明，这里是图书馆，大家都在安静看书。你如果有话想说，可以小声一点，或者我们出去再聊，好吗？练3道劝说题，每题必须有称呼、理由、建议和礼貌语气。
转述老师说：“明天上午八点，请大家带上跳绳到操场集合。”请转告请假的同学。老师说明天去操场。老师让我转告你：明天上午八点，请带上跳绳到操场集合。练5句转述，把“我、你、他”和时间地点改准确。
推荐向同学推荐一本你喜欢的书。我推荐《西游记》，很好看。我推荐《西游记》。这本书情节有趣，人物鲜明，孙悟空勇敢机智，遇到困难也不退缩。读这本书，不但能感受神奇的想象，还能学到坚持和勇气。推荐一本书、一个地方、一个活动，各写3条具体理由。
通知班级周五下午三点在音乐教室排练节目，请写通知。周五排练节目，大家参加。通知：请参加节目排练的同学于周五下午三点到音乐教室集合，请带好节目道具，不要迟到。写3则通知，检查时间、地点、人物、事情、要求是否齐全。
讨论发言你是否赞成小学生每天做家务？我赞成，因为做家务好。我赞成小学生每天做力所能及的家务。因为做家务不仅能减轻父母负担，还能培养责任感。比如我每天整理书桌，学习也变得更有条理。每次发言按“观点+两个理由+一个例子”训练。

实战二

书写与查字典提分实战：操作步骤要清楚

训练点真实题目低分问题正确做法训练任务
音序查字法“湖”字会读不会写，怎样查字典？直接翻页找，步骤说不清。先确定读音hu，再查音序H，找到音节hu，按页码查到“湖”。每天选5个会读不会写的字，说出音序、音节和页码。
部首查字法“鹭”不会读，怎样查？找错部首，笔画数乱数。先找部首“鸟”，数部首笔画，再数剩余笔画，查到读音和意思。每周练10个不会读的字，先找部首再查字典。
形近字辨析清、晴、睛怎样区分？只说长得像，不能解释。清与水有关，晴与太阳有关，睛与眼睛有关。清水、晴天、眼睛。整理5组形近字，每组写“偏旁+意思+组词”。
笔顺纠错“再、医、母”等字常写错笔顺。能写出来但笔顺错误。先看结构，再按规则写：先外后内、先横后竖、先中间后两边等。每周选10个易错字，书空一遍，再田字格写3遍。
错字本听写总错同一个字。只改一遍，没有真正掌握。按“错字-正确字-错因-组词-造句”记录。错字连续三次听写正确后，才从错字本里划掉。

实战三

标点符号提分实战：会判断语气和提示语位置

错句问题诊断正确写法教师点评训练任务
妈妈说：“今天冷吗”？问号位置错误。妈妈说：“今天冷吗？”直接引用的话，句末标点一般放在引号里面。练10句人物语言，检查标点是否在引号内。
我喜欢苹果，香蕉，葡萄。并列词语之间可用顿号。我喜欢苹果、香蕉、葡萄。短小并列词语常用顿号，不用连续逗号。把5个含并列词语的句子改正确。
他说：“我走了。”然后跑出门。句子没有错，但可学习提示语位置。“我走了。”他说完，便跑出门。提示语在不同位置，标点要跟着变，表达也更灵活。同一句话分别写成提示语在前、在中、在后。
春天来了！小草绿了！花儿开了！感叹号过多，语气太满。春天来了，小草绿了，花儿开了。大地像醒来了一样。不是所有开心句都用感叹号，标点要服务语气。把一段感叹号过多的话改得自然。
我想说可是我又不敢说缺少停顿和句末标点。我想说，可是我又不敢说。读到需要换气的地方加逗号，句子结束加句号。每天读一段自己的作文，给需要停顿的地方补标点。

实战四

阅读文体提分实战：不同文体，答题角度不同

文体典型题低分答案满分答法训练任务
记叙文文中的“我”发生了什么变化？我变好了。“我”原来害怕上台，后来在老师鼓励下勇敢演讲，说明“我”变得自信、敢于尝试。读3篇记叙文，每篇找“原来怎样、后来怎样”。
说明文“大约”一词能删去吗？不能，删了不准确。不能。“大约”表示估计，不是准确数字，删去后说法过于绝对，体现了说明文语言的准确性。练5道“能否删去”题，按“词义+删后变化+准确性”作答。
寓言这个故事告诉我们什么道理？要努力。这个故事告诉我们，做事不能只看眼前，要根据实际情况想办法，否则会犯同样的错误。每读一则寓言，写“故事内容+道理+生活例子”。
非连续性文本根据表格提出一条建议。大家要多运动。表格显示，每天运动60分钟的学生体质良好比例更高。因此建议学生每天坚持户外运动，尽量保证60分钟左右。练3道图表题，答案必须有数据依据和具体建议。
科普文本文主要介绍了这种动物的哪些特点？它很厉害。文章主要介绍了这种动物的外形、生活习性和保护自己的方式。读科普文时给每段标注“介绍了什么特点”。

实战五

古诗文提分实战：从字词到画面，再到情感

题型真实题目低分答案满分示范训练任务
画面描绘“碧玉妆成一树高”让你看到怎样的画面？柳树很美。我看到一棵高高的柳树，像用碧玉装扮成的一样，枝条柔嫩翠绿，充满春天的生机。每首诗选一句，按“景物+颜色/动作+感受”说画面。
情感判断《静夜思》表达了诗人怎样的感情？诗人很想家。诗人看到明月，低头思念故乡，表达了漂泊在外时对家乡和亲人的深深思念。背古诗时给每首标注“景物+情感”。
字词解释解释“亡羊补牢”中的“亡”。死亡。这里的“亡”是丢失的意思，“亡羊”就是丢了羊。整理10个古今异义词，每个写两个意思。
句子翻译翻译“学而时习之”。学习很高兴。学习了知识并且按时复习它。翻译时先逐词解释，再整理成通顺现代汉语。
道理启示《守株待兔》告诉我们什么？不要等兔子。这个故事告诉我们不能心存侥幸，不劳而获，要靠自己的努力做事。每篇文言故事写“人物做法+结果+道理”。

实战六

词语专项提分实战：放进语境里辨析和运用

题型真实题目低分问题满分思路训练任务
近义词辨析“安静”和“寂静”哪个适合写教室？只说意思差不多。“安静”适合写有人但不吵的教室；“寂静”更适合写夜晚、山林等非常静的环境。每天选2组近义词，各造两个不同语境的句子。
感情色彩“顽强”和“顽固”能互换吗？都表示坚持。不能。“顽强”是褒义，表示坚强不屈；“顽固”多为贬义，表示不愿改变。整理10组褒贬义词，写清适用对象。
词语搭配选择：提高（水平/方法），改进（水平/方法）。随便搭配。应说“提高水平、改进方法”。搭配要看动词和对象是否习惯。每周积累20个固定搭配，听写时放进句子。
关联词（ ）下雨，（ ）运动会照常举行。填“因为……所以……”。应填“虽然……但是……”，前后意思相反，是转折关系。先判断关系，再填关联词：并列、因果、转折、条件、假设。
成语运用用“津津有味”造句。我津津有味。我津津有味地读着《西游记》，连妈妈叫我吃饭都没有听见。成语造句必须有对象和情境，不能只写半句话。

实战七

单元能力提分实战：每个主题都要产出作品

单元主题训练目标具体任务提分标准
童话与想象想象合理，有冲突和道理。续写一个童话结尾，必须有困难、办法、结果。不是乱编，要能看出人物愿望和故事道理。
自然观察能连续观察变化。连续3天观察一种植物，每天写颜色、形状、大小变化。有前后比较，不只写“很开心”。
人物品质从事例看人物特点。读一篇写人文章，找3处语言或动作，概括人物品质。特点必须有原文依据。
家国情怀理解责任和奉献。读一篇相关课文，写“人物做了什么+我受到什么启发”。不能只喊口号，要联系具体行为。
说明文科普说明清楚、有顺序。介绍一种文具或动物，用至少两种说明方法。有说明对象、特点、顺序和准确词语。
传统文化能查资料、能表达。整理一个传统节日：时间、习俗、传说、我的体验。资料真实，表达有条理，不照搬百科。
成长体验写出前后变化。写一件让我成长的小事，列“原来怎样、后来怎样、明白什么”。重点段写具体，结尾有变化和思考。
综合实践会搜集、筛选、展示信息。围绕“节约用水”做一张资料卡和一段发言稿。有数据、有建议、有自己的观点。

节点 1

字词基础

目标：读准、写对、会理解、会运用。

识字写字

笔画、笔顺、偏旁、间架结构、易错字。

检查：能听写，能组词，能放进句子里。

多音字

长、行、好、假、发、乐、觉、没。

检查：能根据句子意思选择读音。

形近字

清/晴/睛，辨/辩/辫，园/圆，做/作。

检查：能说出不同偏旁和意思。

词语积累

近义词、反义词、词语搭配、四字词语。

检查：能造句，不只会背词。

节点 2

句子运用

目标：把句子写清楚、写具体、写正确。

扩句

谁 + 在哪里 + 怎样地 + 做什么。

例：小鸟飞。变成：可爱的小鸟在蓝天上自由地飞。

缩句

去掉修饰语，保留主干。

例：美丽的花儿开放。缩成：花儿开放。

句式转换

把字句、被字句、反问句、陈述句。

检查：意思不变，表达更准确。

修改病句

重复、缺成分、搭配不当、前后矛盾。

检查：读一遍，看是否通顺。

节点 3

阅读理解

目标：读懂内容，找出依据，按题型答完整。

概括内容

谁 + 做了什么 + 结果怎样。

适合：三四年级记事类阅读。

理解词句

联系上下文，先看前后句，再写文中意思。

答法：这个词在文中指……

人物形象

从语言、动作、神态、心理找依据。

答法：从“……”看出他……

中心思想

主要内容 + 情感或道理。

答法：文章写了……，表达了……

节点 4

习作表达

目标：有内容、有顺序、有细节、有中心。

写人

外貌 + 事例 + 品质。

重点：用一件事表现人物特点。

写事

时间、地点、人物、起因、经过、结果。

重点：经过详写，结尾写感受。

写景

观察顺序 + 景物特点 + 自己感受。

重点：从远到近、从上到下都可以。

作文结构

开头点题，中间详写，结尾扣题。

检查：读完能说出中心是什么。

节点 5

古诗文

目标：会背、会默、懂画面、能说大意。

古诗背诵

先读准，再理解画面，最后默写。

检查：不添字、不漏字、不错别字。

诗意理解

抓景物、人物、动作和情感。

家长问：这句诗让你看到什么画面？

文言字词

之、其、曰、乃、亡、善等常见字。

检查：能结合注释说出意思。

故事复述

读完短文，用自己的话讲出来。

检查：人物、事情、结果说清楚。

节点 6

综合能力

目标：会朗读、会审题、会复述、会复盘。

朗读

读准字音，读出停顿和语气。

检查：不破句，不拖音。

审题

圈出“谁、什么、为什么、怎样”。

检查：先看题，再回文章。

复述

用自己的话说主要内容。

模板：这篇文章主要写了……

错题复盘

错因 + 正确方法 + 下次提醒。

模板：我错在……下次先……
