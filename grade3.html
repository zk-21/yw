三年级起步阶段

从“写一句完整的话”提升到“围绕一个意思写一段话”

三年级是语文能力转折点。孩子不再只做字词句练习，而要开始读懂自然段、找中心句、把观察到的内容写成有顺序、有细节的小片段。

三年级必须形成的 5 个能力

  • 字词能力：能辨析多音字、形近字，能把词语放进句子里使用。
  • 句子能力：能扩句、缩句，能把句子写完整、写具体。
  • 段落能力：知道一段话要围绕一个意思写。
  • 阅读能力：能回到原文找答案，能找中心句和关键词。
  • 表达能力：能写观察片段和简单记事片段。

提优总口诀

句子：谁在什么时候、什么地方、怎样地做什么。

段落：先写中心句，再写具体内容。

阅读：先看题，再回文，圈关键词。

作文：看、听、想、做、感受都要写。

字词基础

从“会读会写”到“会理解会运用”

多音字

长：长大 / 长短

乐：快乐 / 音乐

行：行走 / 银行

觉：觉得 / 睡觉

种：种子 / 种地

练法：每个读音造一句话，不能只背词。

形近字

清 / 晴 / 情 / 睛：清水、晴天、心情、眼睛。

辨 / 辩 / 辫 / 瓣：辨别、辩论、辫子、花瓣。

园 / 圆：花园、圆形。

做 / 作：做事、作业。

练法：看偏旁，想意思，再放进句子检查。

句子专项

三年级提优先从“把句子写具体”开始

扩句

方法：加时间、地点、样子、动作、心情。

例：小鸟飞。变成：一只黄色的小鸟在蓝天上自由地飞。

缩句

方法：保留主干，去掉修饰语。

例：红红的苹果从树上掉下来。缩成：苹果掉下来。

标点

重点：句号、问号、感叹号、冒号、引号。

例：妈妈说：“今天我们去公园。”

句子低分到高分

普通句提优句拔高句
小狗跑来了。一只棕色的小狗摇着尾巴跑来了。一只棕色的小狗从巷口冲出来，尾巴摇得像拨浪鼓，黑亮的眼睛直望着我。
花开了。粉红色的桃花在枝头开了。粉红色的桃花露出了笑脸，在春风里轻轻摇晃。
我很开心。我高兴得跳了起来。我高兴得一蹦三尺高，嘴角怎么也压不下来。
雨下大了。雨点密密地落下来。豆大的雨点噼里啪啦地打在窗户上，像一串急促的小鼓点。

阅读理解

三年级阅读重点：找得到、说得清、答完整

阅读题型与答法

题型答题方法常见失分提优要求
主要内容谁 + 做了什么只抄一句话用自己的话说完整
信息提取回原文圈出答案凭记忆回答答案和原文一致
中心句找能概括整段意思的句子找成细节句判断其他句子是否围绕它写
关键词圈人物、动作、地点、特点圈太多，不会筛选抓最重要的词
修辞作用用了什么 + 写出什么样子只写“很生动”说清楚生动在哪里

阅读短文一：春天来了

春天来了。小草从土里探出头来，花儿绽开了笑脸。小鸟在树上欢快地唱歌，蝴蝶在花丛中翩翩起舞。小朋友们脱下厚厚的棉衣，来到公园里放风筝。天空中飘着五颜六色的风筝，有蝴蝶形的，有老鹰形的，还有金鱼形的。春天真是一个美丽的季节。

基础题：这段话主要写了什么？

提优题：文中写了哪几种风筝？

拔高题：“小草从土里探出头来”好在哪里？

高分答法：用了拟人，把小草当作人来写，写出了小草刚发芽时可爱、有生命力的样子。

阅读短文二：我的小狗

我家有一只可爱的小狗，它叫豆豆。豆豆有一双圆圆的眼睛，像两颗黑宝石。每次我放学回家，它都会摇着尾巴跑到门口迎接我。有一次我生病了，豆豆一直趴在我的床边，用头蹭我的手，好像在安慰我。豆豆真是我的好朋友。

基础题：小狗叫什么名字？

提优题：从哪里可以看出豆豆是“我”的好朋友？

拔高题：“像两颗黑宝石”写出了眼睛什么特点？

高分答法：写出了豆豆眼睛又黑又亮的特点，让小狗的样子更可爱。

作文表达

从一句话到一段话，再到一篇短作文

一段话结构

第一句：写中心意思。

中间：写具体表现，可以写看到、听到、想到、做了什么。

最后：写感受或总结。

例：我家的小猫真可爱。它有圆圆的眼睛，雪白的毛，走起路来轻轻的。每次我回家，它都会跑过来蹭我的腿。我非常喜欢它。

观察作文四步

看：颜色、形状、大小、变化。

听：声音、动静。

想：像什么，有什么发现。

感受：喜欢、惊奇、明白了什么。

古诗文入门

三年级古诗重点：读懂画面，体会简单情感

古诗学习四步

读准：读准字音，不添字、不漏字。

找景物：诗里写了月亮、山水、花草、鸟、雨、雪等什么景物。

想画面：闭上眼睛想一想诗句描写的景象。

说情感：诗人是喜爱、思念、赞美，还是舍不得。

常见诗人情感

思乡：看到月亮，想起家乡和亲人。

喜爱自然：看到春天、花草、鸟鸣，心里高兴。

赞美景物：写山水、田园、季节之美。

惜别：送朋友离开，心里舍不得。

古诗情感答题示例

题型答题方法示例答案
这句诗写了什么画面？景物 + 样子写了月光洒在床前，好像地上结了一层霜的画面。
表达了什么情感？画面 + 情感诗人看到明月，想起家乡，表达了思念家乡的感情。
你喜欢哪句？为什么？喜欢的句子 + 画面 + 感受我喜欢“小荷才露尖尖角”，因为它写出了荷叶刚长出来时可爱的样子。

诗人怎样表达情感

表达方式孩子要看什么例子表达的情感
借景表达看诗里写了什么景物明月、春风、花、鸟、雨思乡、喜爱自然、赞美春天
借动作表达看诗人做了什么举头、低头、望、送、看思念、舍不得、留恋
借画面表达把诗句想成一幅图小荷刚露出尖尖角喜爱新生事物、赞美自然
借关键词表达圈出最能表现心情的词思、愁、喜、怜、惜直接点出诗人的感受
